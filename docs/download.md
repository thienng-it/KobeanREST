# Download KobeanREST

KobeanREST is a local desktop app. Download it, launch it, and use it without a KobeanREST account.

The source of truth for public installers is **GitHub Releases**:

<https://github.com/thienng-it/KobeanREST/releases/latest>

## Stable Downloads

| Platform | Artifact | Notes |
| --- | --- | --- |
| macOS | `KobeanREST-macos-universal.dmg` | Universal DMG for Apple Silicon and Intel where release signing supports it. |
| Windows | `.msi` | Windows installer generated by release CI. |
| Linux | `KobeanREST-linux-x64.AppImage` | Portable Linux desktop artifact. |
| Linux | `KobeanREST-linux-amd64.deb` | Optional Debian package for supported distributions. |

Each release should include `SHA256SUMS.txt` checksums and signed Tauri updater metadata as `latest.json`.

## Checksum Verification

Use the matching line from `SHA256SUMS.txt` before you open an installer.

- macOS or Linux: run `shasum -a 256 <file>` and compare it with `SHA256SUMS.txt`.
- Linux: `sha256sum <file>` works too.
- Windows: run `certutil -hashfile <file> SHA256` in PowerShell or Command Prompt and compare it with `SHA256SUMS.txt`.

## macOS

1. Download the latest `.dmg` from GitHub Releases.
2. Open the `.dmg`.
3. Drag KobeanREST into `Applications`.
4. Launch it locally. No KobeanREST account is required.

## Windows

1. Download the latest `.msi` from GitHub Releases.
2. Open the `.msi` and run the installer.
3. Launch KobeanREST from the Start menu or desktop shortcut.
4. The app works locally without login or registration.

## Linux

For the portable build:

1. Download the latest `.AppImage`.
2. Run `chmod +x KobeanREST-linux-x64.AppImage`.
3. Open the AppImage.

For Debian-based systems:

1. Download the latest `.deb`.
2. Install the `.deb` with your package installer or `sudo dpkg -i <file>`.

## Update Behavior

KobeanREST can check for updates automatically after launch. Update checks use public signed release metadata and do not require login, registration, or a hosted account.

Users can disable automatic update checks from app settings or use `Check now` manually. The app remains fully usable offline, including when update checks fail or are turned off.

## Privacy

KobeanREST stores workspaces, collections, requests, environments, and request history locally. Secrets are stored through the operating-system keychain or encrypted vault support and are excluded from normal exports by default.
